# AI & Data Scientist Complete Roadmap 2025

Your comprehensive guide to becoming an AI & Data Scientist, based on the official roadmap.sh curriculum with 1200+ hours of curated learning content.

## Roadmap Overview

```
┌─────────────────────────────────────────────────────────────┐
│              AI & DATA SCIENTIST LEARNING PATH              │
└─────────────────────────────────────────────────────────────┘

Phase 1: FOUNDATIONS (Months 1-3)
├── Programming (Python, SQL, Git)
├── Mathematics (Statistics, Linear Algebra)
└── Data Analysis (Pandas, NumPy, Visualization)

Phase 2: MACHINE LEARNING (Months 4-6)
├── Supervised Learning (Regression, Classification)
├── Unsupervised Learning (Clustering, PCA)
├── Model Evaluation & Selection
└── Scikit-learn Mastery

Phase 3: DEEP LEARNING (Months 7-9)
├── Neural Networks (PyTorch/TensorFlow)
├── Computer Vision (CNNs, Object Detection)
├── NLP (RNNs, Transformers, LLMs)
└── Specialization Choice

Phase 4: PRODUCTION & CAREER (Months 10-12)
├── MLOps (Docker, Kubernetes, CI/CD)
├── Model Deployment (Cloud, APIs)
├── Portfolio Building (5+ Projects)
└── Job Search & Interviews
```

## Phase 1: Foundations (Months 1-3)

### Week 1-4: Python Programming
**Goal:** Master Python for data science

**Topics:**
- Python syntax, data structures, control flow
- Functions, lambda, decorators
- List/dict comprehensions
- File I/O, exception handling

**Resources:**
- Kaggle Learn Python (Free, 7 hours)
- Google's Python Class (Free)
- "Automate the Boring Stuff" (Free online)

**Practice:**
- 50+ LeetCode Easy problems
- Python challenges on HackerRank

**Milestone:** Build a data processing script

---

### Week 5-6: SQL & Databases
**Goal:** Query and manage databases efficiently

**Topics:**
- SELECT, WHERE, GROUP BY, HAVING
- JOINs (INNER, LEFT, RIGHT, FULL)
- Window functions, CTEs
- Database design basics

**Resources:**
- SQLTutorial.org (Free)
- Mode SQL Tutorial (Free)
- LeetCode SQL problems

**Practice:**
- 30+ SQL problems on LeetCode/HackerRank
- Design a database schema

**Milestone:** Complete a complex multi-table query project

---

### Week 7-8: Git & Version Control
**Goal:** Collaborate and track code changes

**Topics:**
- Git basics (clone, commit, push, pull)
- Branching and merging
- Pull requests and code review
- .gitignore, best practices

**Resources:**
- Pro Git Book (Free online)
- GitHub Learning Lab (Free)
- Atlassian Git Tutorials

**Practice:**
- Create GitHub profile
- Contribute to open source project
- Build first portfolio repo

**Milestone:** GitHub portfolio with 3+ projects

---

### Week 9-10: Statistics & Probability
**Goal:** Understand statistical foundations

**Topics:**
- Descriptive statistics (mean, median, variance)
- Probability theory and distributions
- Hypothesis testing (t-test, chi-square)
- Confidence intervals, p-values

**Resources:**
- Khan Academy Statistics (Free)
- StatQuest YouTube (Free)
- "Think Stats" by Allen Downey (Free PDF)

**Practice:**
- A/B testing analysis
- Statistical significance calculations

**Milestone:** Complete statistical analysis report

---

### Week 11-12: Data Analysis with Pandas
**Goal:** Manipulate and analyze data efficiently

**Topics:**
- DataFrames and Series
- Data cleaning (missing values, duplicates)
- GroupBy and aggregations
- Merge, join, concatenate

**Resources:**
- Pandas documentation
- "Python for Data Analysis" by Wes McKinney
- Kaggle Pandas course

**Practice:**
- Analyze 5 different datasets
- EDA on Kaggle datasets

**Milestone:** Complete Titanic Kaggle competition
